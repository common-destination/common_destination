## Tech Stack

This is a **MERN** project written in **JavaScript** and 
styled with **SASS**.
<br />


**Backend:** NodeJs, Express and MongoDB 
<br />

**Frontend:** ReactJs 
<br />

**Libraries:** moment, node-open-geocorder, geolib, react-datepicker, emailJs

**Tools:** GitHub, VSCode, Slack, Trello, Figma, Prezi
